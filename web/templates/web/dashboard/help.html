Sıkça sorulan sorular

Randevu nasıl eklerim?

Randevular sayfasından veya API POST /api/v1/appointments/ ile ekleyebilirsiniz. Müşteri ve hizmet önceden tanımlı olmalıdır.

Müşteri giriş yapmadan randevu alabilir mi?

Evet. MVP'de müşteri rolü yoktur; herkese açık booking sayfası (/b/<slug>/) veya POST /api/v1/public/book/ ile randevu oluşturulabilir.

API dokümantasyonu nerede?

Proje çalışırken /api/schema/swagger-ui/ adresinden OpenAPI (Swagger) arayüzüne erişebilirsiniz.
